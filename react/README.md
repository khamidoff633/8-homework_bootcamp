# Pokémon React App

## Qanday ishga tushirish kerak

1. React papkasiga o'ting:
```bash
cd react
```

2. Paketlarni o'rnating:
```bash
npm install
```

3. Ishga tushiring:
```bash
npm start
```

## Xususiyatlari

- **Zamonaviy dizayn** - Gradient fon, cardlar, hover effektlari
- **Qidiruv tizimi** - Pokémon nomlari bo'yicha qidiruv
- **Responsive** - Barcha qurilmalarda chiroyli ko'rinadi
- **Type badge'lar** - Har bir Pokémon tushi uchun rangli belgilar
- **Animatsiyalar** - Hover effektlari va silliq o'tishlar

## Struktura

```
react/
src/
  components/
    PokemonCard.js    - Bitta Pokémon kartasi
    PokemonList.js    - Pokémonlar ro'yxati
    SearchBar.js      - Qidiruv paneli
  data/
    pokemons.js       - Pokémon ma'lumotlari
  App.js              - Asosiy komponent
  App.css             - Barcha stillar
```

## Ishlatilgan texnologiyalar

- React 18
- CSS Grid & Flexbox
- Modern JavaScript (ES6+)
- Responsive Design
